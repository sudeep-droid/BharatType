🇮🇳 BharatType – Mangal to Kruti Dev Converter

BharatType is a fast and reliable font conversion tool that converts Unicode (Mangal) text into legacy Kruti Dev format, widely used in government offices, typing exams, and legacy Hindi systems.

🚀 Features
🔄 Accurate Font Conversion (Mangal → Kruti Dev)
⚡ Fast Processing with optimized backend
🌐 Web-based Interface (no installation required)
📄 Supports Large Text Input
🎯 Eliminates Manual Retyping Errors
🧩 Modular architecture (Frontend + Backend separation)
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript / TypeScript
Backend: Python, FastAPI
Deployment: Vercel (Frontend), Render (Backend)
📂 Project Structure
BharatType/
│── backend/          # FastAPI server
│── frontend/         # UI interface
│── utils/            # Conversion logic
│── assets/           # Static files
│── README.md
⚙️ How It Works
User inputs text in Mangal (Unicode Hindi)
Backend processes the text using custom mapping logic
Output is generated in Kruti Dev font encoding
User can copy/download the converted text
🧠 Use Cases
🏛️ Government document formatting
📝 Hindi typing exams (Kruti Dev format)
📚 Legacy system compatibility
👨‍💼 Office work where Kruti Dev is required
🖥️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/sudeep-droid/BharatType.git
cd BharatType
2️⃣ Backend Setup
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
3️⃣ Frontend Setup
cd frontend
npm install
npm run dev
